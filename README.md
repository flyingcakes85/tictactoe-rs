## Build and Run

1. Get the repo

```sh
git clone https://github.com/flyingcakes85/tictactoe-rs
cd tictactoe-rs
```

2. Build/Run

```sh
cargo build            # To build
cargo run              # To run
cargo build --release  # Optimized release build
```
